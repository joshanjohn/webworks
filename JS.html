What is js ?

JavaScript is a programming language that is used for client-side scripting in simple words, it allows to add of functionalities to each HTML element. It enables us to build complex websites. JavaScript is entirely different from Java programming language. JavaScript allows to add functionalities to the web using the DOM(Document Object Model) model. JavaScript files always have a .js file extension.
