What is HTML ?

Hyper Text Markup Language (HTML) was created by Tim Berners-Lee. It is the most fundamental block of a web page, HTML provides meaning and structure to the web, and other technologies are used to add design or functionalities to the web. HTML elements or tags are used to define the layout of the website. HTML file should have .html file extension.

HTML elements

Some HTML tags

  • <html>
    It defines the start/ begining of an HTML document.
  • <body>
    It defines the body of an HTML document.
  • <header>
    Inside head tag we include all the meta tags which are not visible.
  • <main>
    It defines the body of an HTML document.
  • <em>
    It makes any text to italics or emphasis.
  • <title>
    It is used to define the title of our web.
  • <h1>
    It defines/treat text as a Heading 1.
  • <p>
    It defines/treat text as a pragraph.
  • <a>
    It defines/treat text as a hypertext. which act as a text link.
  • <img>
    It is used to show images in our web.
  • <style>
    It is used include css code inside the HTML file.
  • <script>
    It is used include JavaScript code.
  • <link>
    It is used to link things into our html page.
