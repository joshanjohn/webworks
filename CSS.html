What is CSS ?

The full form of CSS is Cascading Style Sheets. which is used to style and design html elements in other words it helps developer to build stylish and user-friendly interfaces. CSS is used for the front end. It is very easy to learn and implement. It is also very easy to style multiple HTML layouts in a single code file. CSS can be externally stored as a .css file format.

reference

Some Css rules

CSS code consists of a selector and its declaration block. The selector tells which HTML element or tag should the design apply. The declaration block includes declarations that have its property and values. The declarations will be inside the curly braces. Each property within the declaration is separated/terminated by a semi-colon.
